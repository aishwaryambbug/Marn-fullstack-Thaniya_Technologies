# Smart Task Management System Using MERN Stack

A full-stack MERN application with:

- User Authentication (JWT)
- Task CRUD Operations
- React Frontend
- Node.js + Express Backend
- MongoDB Database

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- Register/Login
- Add Task
- Update Task
- Delete Task
- Dashboard
